Генерирование кабельного журнала

Для вызова применяется команда КАБЖУРНАЛСПОЛЯМИ

Программа генерирования кабельного журнала обрабатывает блоки, содержащие следующие 7 обязательных атрибута:

-ПОЗ - видимый атрибут, позиционное обозначение кабеля

-ОТКУДА - невидимый атрибут, начало маршрута кабея

-КУДА - невидимый атрибут, конец маршрута кабея

-ДЛИНА - видимый атрибут, суммарная длина прокладки кабеля. Вводится вручную

-ГОФРА - видимый атрибут, длина прокладки кабеля в гофре

-КОРОБ - видимый атрибут, длина прокладки кабеля в коробе

-МАРКА_КАБЕЛЯ - видимый атрибут, марка кабеля, например КВВГЭнг(A)-LS 4x1,5

Наличие других атрибутов никак не сказывается на работе програмы

Пронумеровать рамки листов

Для вызова применяется команда ПРОНУМЕРОВАТЬЛИСТЫ

Программ чувствительна к имени блока. Блоки должны иметь имя "Штамп2" - для первого листа и "Формат3" для последующих листов.
